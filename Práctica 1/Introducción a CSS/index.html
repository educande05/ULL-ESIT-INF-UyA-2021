ORGANIZATE CON TEMPO

Tempo, aplicación diseñada por Tempo Team con el propósito de ayudar a los usuarios con la gestión de su tiempo. Esta aplicación es gratuita y está diseñada para fomentar el estudio y afianzar el contacto entre alumnos y profesores.

Tempo pone a tu disposición todas las herramientas y recursos necesarios para una correcta gestión del tiempo del usuario.
